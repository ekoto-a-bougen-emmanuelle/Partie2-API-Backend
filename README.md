# API Blog Simple

API backend développée avec Node.js, Express, SQLite et Swagger pour gérer des articles de blog.

## Fonctionnalités

- Créer un article
- Lire tous les articles
- Lire un article par ID
- Modifier un article
- Supprimer un article
- Rechercher un article
- Filtrer les articles par catégorie, auteur ou date
- Documentation Swagger

## Technologies utilisées

- Node.js
- Express.js
- SQLite
- Swagger UI

## Installation

```bash
git clone <url-du-depot>
cd blog-api
npm install
npm start
